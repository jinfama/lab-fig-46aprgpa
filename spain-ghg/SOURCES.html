SOURCES.md — Inventario file-level

Tabla canónica file-level de todos los ficheros que viven bajo data/raw/. Cada fila corresponde a UN fichero físico. Se reconstruye con docs/_compute_source_checksums.py (checksums) y se renderiza a SOURCES.html con docs/_render_sources_html.py.

Clasificación: EXT = fuente externa, INT-CAHE = fuente interna del grupo CAHE, INT-DERIVED = derivada en este repositorio.

Cómo se mantiene

  1. Si añades un fichero a data/raw/<source>/, añade fila aquí (manual) y ejecuta python docs/_compute_source_checksums.py para que data/raw/CHECKSUMS.csv quede sincronizado.
  2. Si renombras o mueves, actualiza file path y reejecuta el helper.
  3. Provenance status valores: verified (descarga reproducible con URL+fecha confirmadas), internal (fuente interna CAHE, sin URL pública), derived (construido a partir de otras fuentes).
