SOURCES — Spain Trade Dataset

Última actualización: 2026-05-10 (sesión IMPLEMENT structure Tier 3, post Q01/Q02 PI ratification)

Inventario file-level de toda la primaria bajo data/raw/. Una entrada por archivo atómico con SHA256, citation, status semáforo, used by y download script. Render navegable: SOURCES.html en raíz.

Convenciones:

  • Status: 🟢 verified-by-user / 🟡 agent-verified-cached / 🔴 unverified-or-orphan.
  • Provenance: 1-tier (primaria oficial), 2-tier (compilación publicada), 3-tier (re-compilación secundaria — requiere chain en notas).
  • Used by: scripts del pipeline que consumen el archivo. ORPHAN cuando el path está declarado en 00_config.R pero ningún script lo lee.
  • Download script: si la fuente requiere conversión / extracción, enlace al script en code/extract/<source>/ (decisión D4 forest). Trade actualmente no tiene extractores live; _archive/ contiene helpers OCR históricos no productivos.

Las semillas de este documento provienen de docs/references/inventario_fuentes.md (2026-03-06, v2). Tras el arbitraje PI 2026-05-10 (Q02, opción c), ese inventario y los otros 3 markdown históricos (metodologia_completa.md, metodologia_bilateral.md, metodologia_serie_directa.md) se han archivado en _archive/old_docs_references/ y el contenido vigente queda consolidado aquí (§ entradas file-level + §§ "Material de referencia adicional" más abajo).

Re-generar checksums: python docs/_compute_source_checksums.py. Re-render HTML: python docs/_render_sources_html.py.
